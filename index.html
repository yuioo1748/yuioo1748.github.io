阿庚的自我介紹

  • 個人簡介

    • 我是解庚霖，大學就讀中國文化大學生命科學系(生物相關科系)。

    因為在就學期間有加入教授的實驗室，進行實驗相關的練習與熟悉，
    後來真的也對實驗室內的研究沒有興趣，且想換跑道，但沒有任何方向，
    畢業後有再進修職訓局的「Python 大數據分析」 課程，有學過機器學習、資料分析、網路爬蟲，
    並且在課堂中也學習過MySQL與JAVA，在課程中發現自己對程式很有興趣，並且在資料分析這塊，
    我運用大學平時報告和專題的經驗，使我很快找到了主題，並完成了課題。職訓結束後有自學Git，平時也有做各種程式相關的練習。

    因為看好程式便利性在社會上的發展，也想更精進自己寫程式的能力，所以想從事相關產業， 希望以後能成為專業的軟體工程師。
  • 為什麼想成為前端、後端、或全端工程師？

    • 其實我對前後端的認知就是停留在朋友口頭講，與網路上的資料，真正接觸應該是自己做object-c的ios app練習與這次申請用的網站，
    才對前端有更多的認識，我也自認自己對畫面的處理比較有感覺，且更有興趣。
  • 為了成為軟體工程師，曾經做過什麼努力？

    • 應該就是參加職訓的Python課程，課程一共也就3個月的時間，我覺得在上課期間都算認真， 但對於新手來說初次碰程式語言，語言邏輯還是不太容易過的一個坎，
    也是花了一陣子才比較好上手，期間也考了該考的證照，例如:
      TQC專業程式設計工程師:

    • TQC 人工智慧 : 機器學習 Python 3
    • TQC 網頁資料擷取與分析 Python 3
    • TQC 程式語言 Python 3
    • TQC 軟體開發知識
    • TQC+ Python 大數據分析專業人員證照
    • TQC+ Python 機器學習專業人員證照
  • 如果參與這個計畫，會怎麼安排學習時間？

    • 看這份計畫預計每周要花40~60小時的時間來學習，我會盡我所能完成課題，但對於學習其實我一直都沒有安排計畫的習慣，
    我知道在安排學習時間這件事情上我可能需要有人的幫忙，希望可以從參與這次的計畫學習到良好的時間管理能力。
  • 如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

    • 其實看上次畢業學員分享的經驗中可以大概知道自己會碰到什麼樣的困難，我認為可能會遇到的有程式新手的語言邏輯問題、學習時間安排不當導致的無法在規定時間內完成作業、 學習倦怠期...等。

    語言邏輯的問題不外乎就是多問、多做和查詢網路資源，我相信在每周40~60小時的學習量中，這個問題是可以改善的。而安排時間的問題，可能可以問導師是否有好的方式，或者模仿成效良好的學員。 而學習倦怠問題，藉由適度放鬆與良好的學習時間安排，應該就可以了。
  • 是否有想要加入的軟體公司？為什麼想加入該公司？

    • 其實目前我對要進入什麼公司，沒有明確的目標，我認為可以先接觸並且進入到這個產業，確確實實的完成公司的工作要求，並且領到薪水，我就可以了。

    說真的對未來公司有什麼期望，對於沒什麼工作經驗的我來說，最希望的應該是有良好的工作環境、品質，公司、部門工作氣氛融洽就好了。
  • 想要對我們說的事情？

    • 從職訓局3個月的訓練結束，學習程式到現在正好已經1年了，期間做了很多額外的練習，也投了非常多次的履歷，但是屢屢碰壁，
    我覺得是因為沒有一個好的作品，外加以一個非本科系的人來說，自己練習的方式，成果實在有限，最近剛好看到有人願意辦這個學習計畫，
    我真的很想進入這個產業，希望能給我這次機會，如果可以我會好好把握的，謝謝。
